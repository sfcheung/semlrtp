# Changelog

## semlrtp 0.1.2

CRAN release: 2026-07-19

### Miscellaneous

- Added package doc by `usethis::use_package_doc()`, for easy access to
  the list of help pages.

- Default starting will be used, with `rstart` set, if fixing a
  parameter to zero leads to bad starting values. See
  <https://github.com/sfcheung/semlrtp/pull/66> and
  <https://github.com/sfcheung/semlrtp/issues/65> . Require `lavaan`
  0.6-18 or above. (0.1.1.2)
