# semlrtp: Likelihood Ratio Test P-Values for Structural Equation Models

Computes likelihood ratio test (LRT) p-values for free parameters in a
structural equation model. Currently supports models fitted by the
'lavaan' package by Rosseel (2012)
[doi:10.18637/jss.v048.i02](https://doi.org/10.18637/jss.v048.i02) .
